# To-Do List - Python CLI App

## Description
Application console de gestion de tâches développée en Python.  
Elle permet d’ajouter, modifier, supprimer et organiser des tâches avec sauvegarde automatique.

## Fonctionnalités
- Ajouter une tâche
- Modifier une tâche
- Supprimer une tâche
- Marquer une tâche comme faite ou non faite
- Supprimer toutes les tâches terminées
- Trier les tâches par priorité
- Sauvegarde automatique dans un fichier JSON

## Technologies utilisées
- Python 3
- JSON
- Interface en ligne de commande (CLI)
